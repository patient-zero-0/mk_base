# MK_Base · 数学专业知识库

## 一句话描述

结构化数学知识库，覆盖**数学分析**与**高等代数**，以知识条目为单元，标注依赖关系与跨专业应用场景，面向数学专业学生、跨专业学习者与科研用户，完全公开、永久免费、不收集个人数据。

## 内容范围（MVP）

- **数学分析** Ch1–Ch5：实数与极限、连续、微分、积分、级数
- **高等代数** Ch1–Ch5：多项式、行列式、线性方程组、线性空间、线性变换
- **跨课关联** ≥ 10 条
- **跨专业应用场景** 每门课 ≥ 5 条
- MVP 目标总条目：**≥ 300 条 stable**

不在本期范围：多元函数分析（Ch6）、内积空间与二次型（Ch6）、公式语义搜索、用户账号体系。

## 四种条目类型

| 类型 | ID 前缀示例 | 核心板块 |
|---|---|---|
| 定义 definition | `ANL-DEF-024` | 定义陈述、与相近概念区别、直觉理解、链接 |
| 定理 theorem | `ANL-THM-006` | 条件、结论、几何直觉、证明、常见错误 |
| 例题 example | `ANL-EX-007` | 题目、分析思路、证明、关键技巧、变式 |
| 习题 problem | `ANL-PROB-031` | 题目、提示（折叠）、解答（折叠）、考察点 |

科目前缀：`ANL`（数学分析）、`ALG`（高等代数）、`CROSS`（跨课）

## AI 使用说明

> 本文件供 AI 工具（Claude、Copilot 等）读取，作为对话全局上下文。

- 所有条目必须符合 `CONTRIBUTING.md` 中的模板规范
- 新建条目前先检查 `_meta/todo.md` 中的悬空引用列表
- 公式一律用 LaTeX 源码（`$…$` / `$$…$$`），不使用 Unicode 数学符号
- 条目状态流转：`draft` → `review` → `stable`，不得跳过
- 禁止直接向 `main` 分支提交，必须经过 PR + CI 检查
