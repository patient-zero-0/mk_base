## 题目

设 $f : [0, 1] \to \mathbb{R}$ 连续。证明下列结论：

1. **不动点存在**：若 $f([0, 1]) \subseteq [0, 1]$（即值域在定义域内），则存在 $\xi \in [0, 1]$ 使 $f(\xi) = \xi$。

2. **零点 + 最值组合**：若 $f(0) f(1) < 0$，则 $f$ 在 $(0, 1)$ 内既有零点，又达到最大值与最小值（且最大、最小值异号）。

3. **构造题**：举一个**不连续**的 $f : [0, 1] \to [0, 1]$ 使其**没有不动点**——以此说明第 1 题中"连续"条件不可丢。

## 提示

<details><summary>点击展开提示</summary>

- 第 1 题：构造辅助函数 $g(x) = f(x) - x$。$g$ 连续，$g(0) = f(0) \geq 0$，$g(1) = f(1) - 1 \leq 0$。用 [[ANL-THM-013]] IVT。
- 第 2 题：$f$ 连续 + 端点异号 ⇒ IVT 给出零点；闭区间连续 ⇒ 最值定理 [[ANL-THM-014]] 给出 $\max, \min$ 取得；零点存在 ⇒ $\min \leq 0 \leq \max$，结合端点异号推出严格。
- 第 3 题：构造分段函数，把 $[0, 1]$ 上的每个点都"挪开自己"。例如 $f(x) = \begin{cases} 1 & x < 1 \\ 0 & x = 1 \end{cases}$ ——值域在 $\{0, 1\} \subseteq [0, 1]$，但每个 $x$ 都不是不动点。

</details>

## 解答

<details><summary>点击展开完整解答</summary>

### 第 1 题：不动点存在

**证明：** 令 $g(x) = f(x) - x$，$g : [0, 1] \to \mathbb{R}$ 连续（连续函数的差仍连续）。

**端点值**：

- $g(0) = f(0) - 0 = f(0) \in [0, 1]$（因 $f(0) \in [0, 1]$）。
- $g(1) = f(1) - 1 \in [-1, 0]$（因 $f(1) \in [0, 1]$）。

故 $g(0) \geq 0$ 且 $g(1) \leq 0$。

**情形 1**：$g(0) = 0$ 或 $g(1) = 0$。则 $\xi = 0$ 或 $\xi = 1$ 即为不动点。

**情形 2**：$g(0) > 0$ 且 $g(1) < 0$。
由 [[ANL-THM-013]] IVT（取 $\eta = 0$ 介于 $g(1) < 0 < g(0)$），存在 $\xi \in (0, 1)$ 使 $g(\xi) = 0$，即 $f(\xi) = \xi$。

两种情形都给出不动点。$\blacksquare$

### 第 2 题：零点 + 最值

**证明：**

**零点**：$f$ 在 $[0, 1]$ 连续，$f(0) f(1) < 0$ ⇒ 端点异号 ⇒ 由 [[ANL-THM-013]] 零点定理，存在 $\xi \in (0, 1)$ 使 $f(\xi) = 0$。

**最值取得**：$f$ 在闭区间 $[0, 1]$ 连续 ⇒ 由 [[ANL-THM-014]] 最值定理，存在 $x_M, x_m \in [0, 1]$ 使
$$
f(x_M) = M = \max_{[0, 1]} f, \quad f(x_m) = m = \min_{[0, 1]} f.
$$

**异号性**：

- $\max \geq f(\xi) = 0$，且 $M = f(x_M) \geq f(0)$ 或 $f(1)$，至少有一端 $> 0$ ⇒ $M > 0$（结合 $f(0) f(1) < 0$，必有正值端）。
- 类似地 $m < 0$。

故 $m < 0 < M$，最值**严格异号**。$\blacksquare$

### 第 3 题：不连续的反例（无不动点）

**构造：** 定义
$$
f(x) = \begin{cases} 1 & 0 \leq x < 1, \\ 0 & x = 1. \end{cases}
$$

**验证**：

- 值域：$f([0, 1]) = \{0, 1\} \subseteq [0, 1]$ ✓
- 连续性：$f$ 在 $x = 1$ 处**不连续**——$\lim_{x \to 1^-} f(x) = 1 \neq 0 = f(1)$（跳跃间断）。
- 不动点：
  - 当 $x \in [0, 1)$ 时 $f(x) = 1 \neq x$（因 $x < 1$）；
  - 当 $x = 1$ 时 $f(1) = 0 \neq 1 = x$。
  
  所以**没有任何 $x$ 满足 $f(x) = x$**。

故连续性不可去除——一旦 $f$ 在某处跳跃，构造的不动点可被"跨过"。$\blacksquare$

</details>

## 考察点

- [[ANL-THM-013]] 介值定理（IVT）的**辅助函数构造**——$g = f - x$ 是不动点问题的标配
- [[ANL-THM-014]] 最值定理的**取得性**论证
- 反例构造：用跳跃间断"恰好回避"不动点

## 备注

- **第 1 题是 Brouwer 不动点定理的一维版**——高维（圆盘、球、凸体）需更强工具（拓扑度、组合论），但一维 IVT 足以。
- **第 3 题反例的核心**：连续函数的图像必须"穿过"对角线 $y = x$，但跳跃间断允许图像"绕过"——这是不动点定理对连续性强依赖的几何根源。

## 跨专业应用

- **数值分析**：二分法、Newton 法、Picard 迭代等寻根算法的存在性论证均依赖本题型
- **经济学**：竞争均衡价格 = 超额需求函数的零点；连续性保证均衡存在
- **博弈论**：Nash 均衡 = 最佳响应映射的不动点；高维版本是 Brouwer / Kakutani 不动点定理
