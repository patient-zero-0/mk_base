## 5. 跨专业 applications 候选（每章 ≥ 5 条）

### Ch1 多项式

- 编码理论：CRC / Reed–Solomon 码基于 $\mathbb{F}_q[x]$ 的多项式除法与根
- 密码学：有限域多项式运算（AES 的 $\mathbb{F}_{2^8}$、椭圆曲线）
- 符号计算：CAS 系统（SymPy / Mathematica）的多项式约简与因式分解
- 控制理论：传递函数的极点 = 特征多项式的根，决定系统稳定性
- 数值分析：多项式插值、求根（Newton / 伴随矩阵法）

### Ch2 行列式

- 解析几何：二阶 / 三阶行列式 = 有向面积 / 体积；叉积
- 微积分：Jacobi 行列式（换元）、Wronski 行列式（线性无关判定）
- 图论：矩阵树定理（生成树计数 = 行列式）
- 经济学：投入产出模型的可解性（Leontief 矩阵）
- 物理：线性方程组解的存在唯一（静力学平衡）

### Ch3 线性方程组

- 数值分析：直接法（LU 分解）与迭代法（Jacobi / Gauss–Seidel）
- 电路分析：基尔霍夫定律的节点 / 回路方程组
- 机器学习：最小二乘 / 正规方程 $A^\top A x = A^\top b$
- 优化：线性规划的可行域与基本解
- 计算机图形：仿射 / 投影变换的求逆

### Ch4 线性空间

- 量子力学：态空间（Hilbert 空间）与叠加原理
- 信号处理：函数空间的基展开（Fourier / 小波基）
- 编码理论：线性码作为 $\mathbb{F}_q^n$ 的子空间
- 机器学习：特征空间、核方法的再生空间
- 微分方程：齐次线性 ODE 解集构成线性空间（与 [[ANL-THM-045]] 呼应）

### Ch5 线性变换

- 数据科学：PCA / SVD / 谱聚类（特征值分解）
- 量子力学：可观测量 = 自伴算子，本征值 = 测量值（谱定理）
- 动力系统：线性稳定性由特征值实部决定（与微分方程耦合）
- 图像处理：特征脸（Eigenfaces）、主成分降维
- 微分方程：矩阵指数 $e^{At}$ 与对角化解耦线性方程组
