## 5. 跨专业 applications 候选（每章 ≥ 5 条）

### Ch3 微分

- 数值分析：Newton 法 / 二分法收敛性
- 物理：瞬时速度 / 加速度作为时间导数
- 经济学：边际成本 / 边际效用
- 优化理论：梯度下降的一阶必要条件（Fermat 引理多元推广）
- 工程：误差传播的一阶近似（微分形式）

### Ch4 积分

- 概率论：连续随机变量分布函数 / 期望
- 物理：功 = $\int F \, dx$、电荷 = $\int I \, dt$
- 信号处理：Fourier 变换基础
- 几何：弧长 / 面积 / 体积公式
- 工程：信号能量 $\int |f|^2 \, dt$

### Ch5 级数

- 数值分析：函数计算的 Taylor 截断
- 信号处理：Fourier 级数基础
- 物理：调和振动 / 量子力学态展开
- 计算机图形：泰勒展开做近似（$\sin/\cos$ 表查找替代）
- 数论：$\zeta(s) = \sum 1/n^s$ 引出解析数论
